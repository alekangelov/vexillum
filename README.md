# Vexillum
A fast, open source feature flagging solution. No bloat, no paywalls.

## Overview
Vexillum is an open source feature flagging solution. It's really open source, no gated features. It allows you to manage feature flags and experiments in your applications. Provides automation capabilities to help you manage your feature flags more effectively.

## Features
- Flag management: Create, update, and delete feature flags.
- Targeting rules: Define rules to control who sees which features.
- Automation: Automate flag management based on predefined conditions.
- SDKs: Client libraries for various programming languages to integrate with your applications.
- Dashboard: A web-based interface to manage your feature flags and view analytics.
- REST API: Programmatic access to manage feature flags and configurations.
- Open Source: Fully open source with no gated features.
- Fast and lightweight: Designed for performance and minimal resource usage.
